# 宅建士合格アプリ

2026年度宅建試験向けの学習管理PWAです。

## 公開URL

**https://kitaha125-wq.github.io/takken/**

## 機能

画面上部のタブから各機能に移動できます。

| タブ | 機能 |
|------|------|
| 🏠 ホーム | 試験までの日数、今日のタスク、学習統計 |
| 📅 計画 | 学習カレンダー、28日間の計画表 |
| 📋 進捗 | 権利関係82講義の進捗トラッカー |
| ✏️ 問題集 | ○×問題、忘却曲線復習、スキップ・問題修正 |

## 以前のリポジトリについて

`kitaha125-wq/takken-2026-` にあったアプリの後継です。  
最新版はこのリポジトリ（`takken`）で更新しています。

## ローカルで開く

```bash
python3 -m http.server 8000
```

ブラウザで http://localhost:8000 を開いてください。
