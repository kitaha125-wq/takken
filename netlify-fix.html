📱 NetlifyがiPhoneで開けない理由

あなたのNetlifyは「古い版」のままです。

スマホでは重すぎて起動できません。
1回だけ更新すれば、同じURLで開けるようになります。

※ 進捗データは消えません（同じURLのまま中身だけ新しくします）

方法A：Git連携（いちばん簡単・おすすめ）

① iPhoneのSafariで app.netlify.com にログイン
② サイト reliable-mousse-e94d97 をタップ
③ Site configuration → Build & deploy
④ Link repository → GitHub → kitaha125-wq/takken を選ぶ
⑤ Branch: main → Deploy
⑥ 2〜3分待ってから開く：
reliable-mousse-e94d97.netlify.app/index.html

方法B：今すぐ勉強したい（更新を待たない）

GitHub版（スマホ対応済み）を使います。JSONを1回復元してください。

バックアップ復元を開く アプリを開く（更新版）

更新できたか確認
